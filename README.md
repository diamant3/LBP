# LBP

Bytepusher VM implementation using Lua/Love2D

## Samples

![LBP showcase](LBP.gif)

## Building

**Requirements**

- [Love2D](https://love2d.org/)
- [Lua](https://www.lua.org/download.html)

### Run

```
love . "roms/ROM.bp"
```
> **Note**
> If inside the LBP folder

or

```
love LBP "roms/ROM.bp"
```
> **Note**
> If outside the LBP folder
